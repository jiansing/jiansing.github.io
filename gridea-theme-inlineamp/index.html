Gridea Theme InlineAMP (v1.0.0)

Jiansing 2020-03-14

移植于wordpress的同名主题 InlineAMP 。（原主题Github）

InlineAMP 是一款纯AMP主题，采用了经典的两栏博客布局，适合喜欢极简设计、比较在意页面打开速度的博主使用。

仓库下载：https://github.com/jiansing/gridea-theme-inlineamp

继承了原主题的一些特点：

  • 纯AMP主题（极致速度体验）
  • 移动端/PC端响应式布局（Flex布局）
  • Pure CSS design（无图设计）
  • 极简&科技风格（简约但不简单）

目前实现的自定义配置：

顶部标语

可以设置主/副标语，将显示在每个页面的顶端

个人信息

在这里填写作者名称

友链

目前仅能在侧边栏自定义友链

Google Analytics

可以添加Google Analytics

语言

用于设置 html lang 属性，默认设置为简体中文(zh-Hans)，一般用默认设置即可

使用提示

由于用了AMP技术，需要遵循AMP HTML规范 ，插入图片需要使用特殊的方式插入
